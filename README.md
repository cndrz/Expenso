# EXPENSO

A raw, minimalistic, AI-powered expense tracker. Built with a brutalist design philosophy. No bloat. Just data and actionable insights.

## FEATURES
- **MAGIC_ENTRY**: Parse natural language into structured data using Groq AI.
- **VOICE_INPUT**: Hands-free transaction logging using the Web Speech API. 
- **AI_COACH CHATBOT**: A contextual AI chatbot that reads your actual spending history to give you direct, highly constructive financial advice.
- **PDF_DOSSIER**: Instantly generate a highly-styled brutalist PDF report containing your total net worth, AI analysis, and a raw transaction ledger.
- **SPEND_BLOCKS**: A brutalist bar chart visualizing your spending limits and identifying areas where you are "BANKRUPT".
- **CLOUD_SYNC**: Fully backed by Supabase. Secure authentication (Email, Password, Username) and real-time syncing of your expenses across all devices using PostgreSQL Row Level Security.
- **INVERT_REALITY**: A seamless custom dark mode implementation that swaps core CSS variables.
- **DATA_VAULT**: Export and Import your JSON data manually if you prefer local backups.


## STACK

- SUPABASE (Auth & PostgreSQL)
